

## Project Name

**Greenden-tailwind(Responsive Plant Store Website)**

## Project Description

Greenden is a modern and responsive plant-themed website developed using **HTML** and **Tailwind CSS**. The project is designed to showcase plants, gardening products, and eco-friendly content in a clean and attractive user interface. The website focuses on responsive web design, ensuring a smooth experience across mobile, tablet, and desktop devices.

The project includes a responsive navigation bar, product sections, customer-friendly layouts, and interactive styling using utility classes from Tailwind CSS. Greenden demonstrates how modern frontend technologies can be used to create visually appealing and user-friendly web pages.

### Features of the Project

* Responsive navigation bar for mobile and desktop screens
* Modern and clean UI design
* Product display section for plants and gardening items
* Interactive hover effects and styling
* Responsive layouts using Flexbox and Tailwind utilities
* Sticky navigation bar for better user experience
* Mobile-friendly hamburger menu icon

### Technologies Used

* **HTML5** – For structuring the webpage
* **Tailwind CSS** – For responsive styling and layout design
* **SVG Icons** – For navigation icons and visual enhancement

## HTML Tags Used in the Greenden Project

### Structure Tags

* `<html>` – Defines the root of the HTML document
* `<head>` – Contains metadata and title information
* `<meta>` – Defines character set and responsive viewport settings
* `<title>` – Sets the webpage title
* `<body>` – Contains visible webpage content

### Navigation Tags

* `<nav>` – Creates the navigation bar section
* `<h1>` – Displays the website title/logo (GREENDEN)
* `<ul>` – Creates an unordered list for navigation items
* `<li>` – Represents list items in the menu
* `<a>` – Creates clickable navigation links

### Icon Tag

* `<svg>` – Displays scalable vector icons (hamburger menu icon)
* `<path>` – Defines the shape/path of the SVG icon

### Styling Concepts Used (Tailwind CSS Classes)

* `bg-green-200` – Background color
* `p-3` – Padding
* `flex` – Flexbox layout
* `items-center` – Align items vertically center
* `justify-between` – Space between elements
* `shadow-xl` – Large shadow effect
* `sticky` – Sticky navigation behavior
* `top-0` – Stick to top position
* `z-10` – Layer positioning
* `hidden` – Hide elements
* `md:flex` – Show flex layout on medium screens and above
* `md:hidden` – Hide elements on medium screens and above
* `gap-5` – Space between items
* `text-gray-600` – Text color
* `hover:underline` – Underline effect on hover
* `text-2xl` – Font size utility
